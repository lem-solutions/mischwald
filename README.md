# Mischwald

Implementierung der Spiellogik des Kartenspiels „Mischwald“.

Bisher gibt es keine Tests und eine Karte fehlt(_Fuchs du hast die Karte gestohlen, gieb sie wieder her!_). Bugs im Code gibt es wahrscheinlich genau so viele wie im Wald ;)

Geplant ist die Implementierung von Spieler-KIs(sowohl algorithmisch als auch ML) um einige Statistiken(welche Karten werden von den KIs bevorzugt, wie viele Punkte werden durchschnittlich erreicht, usw.) zu erfassen.
